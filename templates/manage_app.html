Quick Start (C# SKYLINE SDK)

1. Download the C# SDK

Click Download C# SDK above — the file comes pre-configured with your app's credentials.

2. Add to your C# Project

Drop SKYLINE.cs into your project. It targets .NET Framework 4.8+ and uses System.Net.WebClient.

3. Initialize & Authenticate

// ── Your credentials (already filled in the downloaded file) ──
var SKYLINEApp = new SKYLINE.api(
    name:    "{{ app.name }}",
    ownerid: "{{ app.owner_id }}",
    secret:  "{{ app.secret_key }}",
    version: "{{ app.version if app.version else '1.0' }}"
);

// ── Initialize (call once at startup) ──
SKYLINEApp.init();

// ── Login with username + password ──
SKYLINEApp.login("username", "password");
if (SKYLINEApp.response.success)
    Console.WriteLine("Welcome, " + SKYLINEApp.user_data.username);
else
    Console.WriteLine("Error: " + SKYLINEApp.response.message);

// ── Register with a license key ──
SKYLINEApp.register("username", "password", "LICENSE-KEY");

// ── License-only login (no username/password) ──
SKYLINEApp.license("LICENSE-KEY");
if (SKYLINEApp.response.success)
    Console.WriteLine("Authenticated!");

4. Point to Your Server

The downloaded SDK already points to {{ api_url }}/ — no manual changes needed.
